## 📌 Overview

This project showcases a **complete end-to-end expense automation pipeline** built using modern no-code tools and analytics platforms.

It integrates:

* 📥 Data Collection (**Google Forms**)
* 🗂️ Data Storage (**Google Sheets**)
* ⚙️ Workflow Automation (**n8n**)
* 📧 Notifications (**Gmail**)
* 📊 Visualization (**Power BI**)

👉 Result: A **real-time financial monitoring system** for smarter decision-making.

## 🔄 End-to-End Flow Explanation

1. 📝 User submits expense via Google Form
2. 📊 Data is automatically stored in Google Sheets
3. ⚙️ n8n detects new entry and triggers workflow
4. 📧 Email notification sent instantly
5. 📈 Power BI dashboard updates in real-time
